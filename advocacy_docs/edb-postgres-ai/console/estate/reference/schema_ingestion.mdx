---
title: Schema Ingestion into the EDB Postgres AI Console Estate
navTitle: Schema Ingestion
description: The schema's that can be ingested into the EDB Postgres AI Console Estate.
---


With Schema Ingestion enabled, the following information gets sent to your Estate:

* Schema names
* Table names
* Table types
* Column names
* Column types
* Column default values
* Column nullablity
* Table description (from comments)
* Column description (from comments)

!!! Note
The schema ingestion process is secure and doesn't collect any data from the tables themselves. If, though the schema information contains personally identifying information (PII), such as customer names as a column name, then that PII will be ingested into the Estate.
!!!
