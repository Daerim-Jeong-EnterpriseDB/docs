Chemaxon is a leader in the field of cheminformatics, providing chemical software development tools to the pharmaceutical and biotechnology industries. Chemaxonâ€™s offerings consist primarily of tools for visualizing data, drawing molecules, and searching chemical databases.

The Chemaxon JChem PostgreSQL Cartridge integrates chemical structure handling with EDB Postgres Advanced Server and PostgreSQL through extensions to PostgreSQL's native SQL language. It enables scientists to create, manage, and analyze chemical structures and related nonchemical data in their Postgres database. JChem users can search data by superstructure, substructure, duplicate, and full fragment (exact fragment) search.
