Thales CipherTrust Manager offers an industry-leading enterprise key management solution to help organizations centrally manage encryption keys, provide access control, and configure security policies. Thales CipherTrust Manager can be used in conjunction with EDB Postgres Advanced Server version 15.2 and EDB Postgres Extended Server 15.2 and above, Transparent Data Encryption (TDE) capabilities to further secure your sensitive data at-rest.

Thales CipherTrust Manager manages key lifecycle tasks such as generation, rotation, destruction, import and export, and provides role-based access control to keys and policies. Thales CipherTrust Manager can be utilized in both virtual and physical appliances that integrate with FIPS 140-2 compliant third-party Hardware Security Modules (HSM) to securely store keys.
