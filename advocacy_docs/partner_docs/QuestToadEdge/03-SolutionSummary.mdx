Quest Toad Edge is a lightweight and reliable desktop database toolset that streamlines development and management tasks for EDB Postgres Advanced Server and EDB Postgres Extended Server. Its flexibility lies in being built on Java and its ability to work with both Windows and Mac operating systems. Toad Edge supports coding, editing, schema compare and sync, and DevOps CI processes, so you can manage EDB Postgres Advanced Server and EDB Postgres Extended Server.
