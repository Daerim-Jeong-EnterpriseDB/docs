## Create the three remote repositories

1. Log in to your JFrog Artifactory instance using your credentials.

2. Go to the **Admin** tab and select **Repositories** from the dropdown menu.

3. Select the **+ Add Repositories** button.

4. Choose **Remote** as the repository type.

5. Fill out the required information for the repository.
   * Repository Key: Unique identifier for the repository.
   * For x86_64: ebd-enterprise
   * For noarch: edb-enterprise-noarch
   * For SRPMS: edb-enterprise-src
   * Package Type: RPM

         -  Remote Repository URL: URL of the remote repository you want to proxy.
For x86_64: https://downloads.enterprisedb.com/<EDB_CS_TOKEN>/enterprise/rpm/el/8/x86_64
For noarch: https://downloads.enterprisedb.com/<EDB_CS_TOKEN>/enterprise/rpm/el/8/noarch
For SRPMS: https://downloads.enterprisedb.com/<EDB_CS_TOKEN>/enterprise/rpm/el/8/SRPMS
