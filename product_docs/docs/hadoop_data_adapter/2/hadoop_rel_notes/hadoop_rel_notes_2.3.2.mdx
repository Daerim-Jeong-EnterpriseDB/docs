navTitle: Version 2.3.2

Released: 17 Jul 2024

Enhancements, bug fixes, and other changes in Hadoop Foreign Data Wrapper 2.3.2 include:

| Type        | Description                                                                                                              |
|-------------|--------------------------------------------------------------------------------------------------------------------------|
| Enhancement | Added support for PostgreSQL 17 and upcoming EDB Postgres Advanced Server 17.                                            |
| Feature     | Added `enable_limit_pushdown` GUC to control LIMIT/OFFSET push-down, at the session level.                               |

!!!important Deprecation
   In this version, we deprecate support for databases on EDB Postgres Advanced Server 11.
   We recommend upgrading to the lastest available version.
