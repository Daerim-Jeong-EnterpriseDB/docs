New features, enhancements, bug fixes, and other changes in PEM 9.2.0 include:

| Type        | Description                                                                                      |
| ----------- | -------------------------------------------------------------------------------------------------|
| Bug fix     | Fixed the "Replication Time Lag" graph to display the status as stopped if the server is down/offline in the Streaming Analysis dashboard. |
| Bug fix     | Removed the agent status from Failover Manager Dashboard under `efm_cluster_node_status`. |
| Bug fix     | Fixed an "unknown error" while updating/creating the server from the PEM web client. |
| Bug fix     | Fixed PUT server REST api to return valid message for invalid parameter and removed unwanted params.
| Bug fix     | updated cryptography package to secure version
| Bug fix     | updated webpack(NPM) package to secure version
| Bug fix     | Fixed error caused by increase/decrease in number of params when changing alert templates
| Bug fix     | fixed the 'Static object initialization fiasco' issue caused during initialising the BARMAN singleton object at the startup
| Bug fix     | Resolved the error "Login Group/Roles 500 error: UNION types char and text cannot be matched" while trying to open the depenedents tab for any role with PG/EPAS15 as backend
| Bug fix     | Provided usefule message to user instead of default python error if token is invalid
| Bug fix     | fixed extension name warning, even after providing valid extension name while creating probe
| Bug fix     | Fixed the error in --config-file switch while trying to register barman server in PEM using cli
| Bug fix     | Document replacing PEM SSL self-signed certificates with those from a trusted third-party CA
| Bug fix     | fixed missing favicon icon for help docs and API link
| Bug fix     | Fixed warn message while deleting folder in Capacity Manager > Manage Template
| Bug fix     | Fixed Tunning Wizard report: Number of server selected is not matching with the servers displayed in the GoTo dropdown
| Bug fix     | Set default time for the end of the sample set in performance diagnostics to be shown as "Now"
| Bug fix     | Alert template : client side validation for alert creation for parameters section
| Bug fix     | Add dateTime empty validations for logAnalysis
| Bug fix     | Fixed chart not updating when period is changed from 'Historical days and threshold' to 'Start time and end time'
| Bug fix     | Added validation for Log Rotation in Audit manager
| Bug fix     | Fixed the missing Extension level metrices while createing chart
| Bug fix     | Remove parameter validation for system alerts
| Bug fix     | fixed PEM overwrites user supplied certificate & key during agent registration
