---
title: "PG_NAMESPACE"
---

The `PG_NAMESPACE` view describes namespaces, which are the structures underlying SQL schemas. Each namespace can have a separate collection of relations, types, and so forth without name conflicts. In addition to the information in the [PostgreSQL pg_namespace description](https://www.postgresql.org/docs/current/catalog-pg-namespace.html), EDB Postgres Advanced Server provides the following information:

| Name               | Type    | Description                          |
| ------------------ | ------- | ------------------------------------ |
| nspbodysource      | text    |                                      |
| nspcompoundtrigger | boolean |                                      |
| nspforeignserver   | oid     |                                      |
| nspheadsrc         | text    |                                      |
| nspobjecttype      | oid     |                                      |
| nspparent          | oid     |                                      |
| nspremoteschema    | text    |                                      |
| nspsecdef          | boolean |                                      |
