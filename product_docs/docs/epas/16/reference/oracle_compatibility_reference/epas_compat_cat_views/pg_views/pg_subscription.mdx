---
title: "PG_SUBSCRIPTION"
---

The `PG_SUBSCRIPTION` view provides information about all existing logical replication subscriptions. In addition to the information in the [PostgreSQL pg_subscription description](https://www.postgresql.org/docs/current/catalog-pg-subscription.html), EDB Postgres Advanced Server provides the following information:

| Name                | Type    | Description                                                 |
| ------------------- | ------- | ----------------------------------------------------------- |
| subpasswordrequired | boolean | Specifies whether the connection requires a password        |
| subrunasowner       | boolean | Specifies whether replication run as the subscription owner |
