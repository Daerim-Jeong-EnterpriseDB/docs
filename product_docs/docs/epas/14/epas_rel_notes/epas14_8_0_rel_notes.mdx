---
title: "Version 14.8.0"
---

EDB Postgres Advanced Server 14.8.0 includes the following enhancements and bug fixes:

| Type           | Description                                                                                                                         |
| -------------- | -------------------------------------------------------------------------------------------------------------------------------------| 
| Upstream merge | Merged with community PostgreSQL 14.8. See the [PostgreSQL 15 Release Notes](https://www.postgresql.org/docs/release/14.8/) for more information.                           | 
| Enhancement    | SQL Profiler and Index Advisor are separate extensions and can be download separately from the [EDB Repos](https://repos.enterprisedb.com/). | 
| Bug fix        | Profile: Don't dump "PASSWORD EXPIRE AT" when the password status is not expired to avoid marking the user account as expired after an upgrade. |
| Bug fix        | Fix the password profile behavior after the password grace time has changed meanwhile. |
| Bug fix        | Fix unexpected error for `edb_enable_pruning` parameter. [Support ticket: #89863] |
| Bug fix        | Fix `cancel_load()` for interrupted by `Ctrl-c`(SIGINT) in EDB*Loader. [Support ticket: #88734] |
| Bug fix        | Set correct object descriptions for redaction policy to make pg_dump work cleanly with --clean and --if-exists options. |
| Bug fix        | Fix pg_dump to dump password verify function for the user profile.  |
| Bug fix        | Fix assertion failure while terminating the process within the autonomous transaction. |
| Bug fix        | Fix corner-case uninitialized-variable issues in SPL. |
| Bug fix        | Fix memory leakage in anonymous blocks that use cast expressions. [Support ticket: #88816] |
| Bug fix        | Fix EOF error from newly created dynamic partition when using edb_partition. |
| Bug fix        | Don't dump permissions of sequences we are not creating while using pg_dump. [Support ticket: #89466] |
| Bug fix        | Fix query jumble to take edb_wrap into consideration while generating queryid. |
| Bug fix        | Fix compiler warning in userenv(). |  |
