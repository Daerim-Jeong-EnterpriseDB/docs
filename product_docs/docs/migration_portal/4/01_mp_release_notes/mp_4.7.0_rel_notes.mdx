Released: 15 Nov 2023

New features, enhancements, bug fixes, and other changes in Migration Portal 4.7.0 include the following:

|     Type     |                                                                                                                            Description                                                                                                                             |
| ------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Enhancement  | Added support for EDB Postgres Advanced Server version 16.                                                                                                                                                                                                         |
| Security fix | Updated the snakeyaml library used by Migration Portal to address the security vulnerabilities identified in [CVE-2018-1000613](https://nvd.nist.gov/vuln/detail/CVE-2018-1000613) and [CVE-2022-25857](https://nvd.nist.gov/vuln/detail/CVE-2022-25857).          |
