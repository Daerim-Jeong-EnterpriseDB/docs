## Set up the EDB repository

In the commands that follow, replace `<TOKEN>` with your token.

1. Get the token to access the EnterpriseDB Beta repository. 

1. For RHEL, SUSE, and so on, run the following command:

   ```shell
   curl -1sLf \'https://downloads.enterprisedb.com/<TOKEN>/enterprise_beta/setup.rpm.sh' \
     | sudo -E bash
   ```

   For Debian and Ubuntu, run the following command: 

   ```shell
   curl -1sLf \'https://downloads.enterprisedb.com/<TOKEN>/enterprise_beta/setup.deb.sh' \
     | sudo -E bash
   ```

## Install EDB SQL Assess

Once the repository is configured, install the software packages using one of the following commands. 

For RHEL 8 and 9:

```shell
sudo dnf -y install edb-sqlassess
```

For Debian and Ubuntu: 

```shell
sudo apt-get -y install edb-sqlassess
```

For RHEL 7 and CentOS 7: 

```shell
sudo yum -y install edb-sqlassess
```

For SLES: 

```shell
zypper -n install edb-sqlassess
```
