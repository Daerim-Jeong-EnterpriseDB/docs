---
title: Returning the Lasso report
---

The Lasso report on a single instance ends
when you return the following information
to your contact person at EDB:

1. The TAR files from phase 1 by uploading them in the Portal
   using the **Upload report** menu or as a specific ticket attachment
2. A document containing the answers to the questions from phase 2
   by uploading it in the Portal in the **My Documents** section
