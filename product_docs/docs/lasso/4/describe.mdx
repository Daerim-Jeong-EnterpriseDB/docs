## GNU/Linux Operating System
### barman crontab/cron (`barman_crontab_cron`)

/etc/cron.d/barman, if exists.

Report output:

  * file `/linux/barman_cron.data`: Content of /etc/cron.d/barman, if exists
  * file `/linux/barman_crontab.data`: Output from barman crontab -l, if barman user

**Depth:** surface


**Security Impact:** *low* &mdash; 
May have entries in crontab/cron with sensitive data
