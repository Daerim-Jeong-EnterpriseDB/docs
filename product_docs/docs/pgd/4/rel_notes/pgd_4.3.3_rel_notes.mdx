EDB Postgres Distributed version 4.3.3 is a patch release of EDB Postgres Distributed 4, which includes bug fixes for issues identified in previous versions.

!!! Note
This version is required for EDB Postgres Advanced Server versions 12.14.18, 13.10.14, 14.7.0 and later.
!!!

| Component | Version |  Type        |                                                   Description                                                           |
| --------- | ------- | ------------ | ------------------------------------------------------------------------------------------------------------------------|
| CLI       | 1.1.2   | Enhancement  | Upgrade `pgconn` lib version to address `connect_timeout` issue in go driver (BDR-3785) |
| CLI       | 1.1.2   | Bug fix      | Set the correct defaults for verbose flag |
| Utilities | 1.2.0   | Enhancement  | Add support for pg_upgrade of BDR 3.7 |
| Utilities | 1.2.0   | Bug fix      | Remove the pre-upgrade step which alters the CONNECTION LIMIT to 0 at database level (BDR-4159) |
