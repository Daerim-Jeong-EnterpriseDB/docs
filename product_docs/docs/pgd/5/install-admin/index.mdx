This section of the manual covers how to install or deploy EDB Postgres Distributed and how to administer it.

There are four main ways to install or deploy PGD:

* [Manual Installation and Administration](admin-manual): How to manually install and administer EDB Postgres Distributed on a set of servers. 


* [Trusted Postgres Architect (TPA)](admin-tpa): How to use TPA to deploy and administer EDB Postgres Distributed to a Docker environment, Linux hosts, or AWS.


* [EDB Postgres Distributed for Kubernetes](admin-kubernetes): How to deploy and administer EDB Postgres Distributed to a Kubernetes environment.


* [EDB BigAnimal](admin-biganimal): How to create and administer EDB Postgres Distributed on the EDB BigAnimal service.
