You can manually deploy and administer EDB Postgres Distributed. 

* [Installing](installing) works through the steps needed to:
  * Provision hosts
  * Install Postgres
  * Configure repositories
  * Install the PGD software
  * Create a cluster
  * Check a cluster
  * Configure PGD proxies
  * Install and use PGD CLI

The example cluster used to show the installation process will be used in future examples.
