### Limitations TDE imposes on standard Postgres functionality

(Current: create_database/ - prohibits use of FILE_COPY strategy)

-------- current
If transparent data encryption is enabled, the `FILE_COPY` strategy cannot be used in `strategy` parameter with `create database`.

[PostgreSQL CREATE DATABASE documentation](https://www.postgresql.org/docs/15/sql-createdatabase.html)
