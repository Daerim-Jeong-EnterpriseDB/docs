## Testing Transparent Data Encryption

To run the tests with transparent data encryption enabled, set the environment variable `PG_TEST_USE_DATA_ENCRYPTION`, for example:

```shell
make check PG_TEST_USE_DATA_ENCRYPTION=1
```
