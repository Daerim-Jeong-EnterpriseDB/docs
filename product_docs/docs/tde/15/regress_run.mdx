## Testing Transparent Data Encryption

To run the tests in [single user mode](/tde/latest/single_user/) with transparent data encryption enabled, set the environment variable `PG_TEST_USE_DATA_ENCRYPTION`. For example:

```shell
make check PG_TEST_USE_DATA_ENCRYPTION=1
```
