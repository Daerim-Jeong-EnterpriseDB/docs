Released: 17 Jul 2024

Enhancements, bug fixes, and other changes in MongoDB Foreign Data Wrapper 5.5.2 
include:

| Type        | Description                                                                                                                      |
| ----------- | -------------------------------------------------------------------------------------------------------------------------------- |
| Enhancement | Added support for PostgreSQL 17 and upcoming EDB Postgres Advanced Server 17.                                                    |
| Deprecation | Removed support for legacy drivers.                                                                                              |

!!!note Deprecation
   In this version, we deprecate support for databases on EDB Postgres Advanced Server 11.
   We recommend upgrading to the latest available version.
